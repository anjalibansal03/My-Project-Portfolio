PROJECTS

EMPLOYEE MANAGEMENT SYSTEM

The Employee Management System is a full-stack web application developed using HTML, CSS, JavaScript for the front end, Django as the backend framework, and SQLite as the database. It enables users to add, update, delete, and view employee records with fields like name, email, department, role, and salary. Django handles routing, form processing, and server-side logic, while SQLite stores persistent data. The front end interacts with Django views and templates to render dynamic content, and JavaScript enhances interactivity.

STUDENT MANAGEMENT SYSTEM

The Student Management System is a full-stack web application built using HTML, CSS, and JavaScript for the front end, Django as the backend framework, and SQLite as the database. This system allows users to add, update, delete, and view student records with key fields such as name, roll number, class, email, and performance details. Django handles server-side operations including form processing, data validation, and routing, while SQLite ensures reliable and lightweight data storage. The front end is styled with CSS for a responsive UI, and JavaScript is used to enhance interactivity and provide real-time feedback. This project demonstrates the integration of front-end and back-end technologies to create a complete and functional student management solution.
